# Security Policy

## Reporting a vulnerability

If you discover a security issue in this project, please report it privately
rather than opening a public issue.

**How to report:**
- Use [GitHub Private Vulnerability Reporting](https://github.com/dkeeno/terraform-mcp-server/security/advisories/new), or
- Reach out via my [GitHub profile](https://github.com/dkeeno)

Please include:
- Affected version / commit
- Steps to reproduce
- Impact assessment

**Expected response:** within 5 business days.

## Supported versions

Only the latest release on the `main` branch is actively supported. Older
versions are provided as historical artifacts and do not receive patches.
